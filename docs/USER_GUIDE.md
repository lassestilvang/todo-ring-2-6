# TaskPlanner User Guide

Welcome to TaskPlanner! This guide will help you get the most out of your task management application.

## Getting Started

### Creating Your First Task

1. Click the input field at the top: "What needs to be done? (⌘K)"
2. Type your task (e.g., "Prepare presentation for tomorrow")
3. Press Enter or click the + button

**Pro tip:** Use natural language!
- "Call John tomorrow at 2pm" → Creates task with date
- "Meeting with high priority" → Sets priority to high
- "Submit report by Friday" → Sets deadline

### Navigating Views

| View | Shortcut | Description |
|------|----------|-------------|
| List | L | Traditional task list |
| Board | B | Kanban-style columns |
| Calendar | C | Calendar view |
| Gantt | G | Timeline and dependencies |

### Keyboard Shortcuts

| Shortcut | Action |
|----------|--------|
| ⌘K / Ctrl+K | Focus search |
| ⌘N / Ctrl+N | New task |
| ? | Show help |
| Escape | Dismiss dialogs |
| A | Select all tasks |
| Shift+A | Clear selection |

## Core Features

### Task Management

**Creating Tasks:**
- Click the input field or press ⌘K
- Use natural language processing
- Set due dates, priorities, and descriptions

**Completing Tasks:**
- Click the empty circle to mark complete
- Completed tasks move to the bottom
- Strikethrough indicates completion

**Editing Tasks:**
- Click any task to open details
- Modify title, description, dates, priority
- Add subtasks, labels, and attachments

### Goals

Track your productivity with goals:

1. Click the "New Goal" button in the Goals section
2. Set a target (e.g., "10 tasks completed")
3. Choose period: Daily, Weekly, Monthly, Yearly
4. Track progress with the visual indicator

### Habits

Build streaks with habit tracking:

- Tasks marked as habits show streak counters
- Complete daily to maintain your streak
- View your progress calendar

### Dependencies

Visualize task relationships:

- Tasks blocked by incomplete dependencies appear in amber
- Circular dependencies are highlighted in red
- Drag to create new dependencies

### Time Blocking

Schedule tasks on a timeline:

- Switch to Gantt view for time blocking
- Drag tasks to specific time slots
- See resource allocation

## Collaboration

### Sharing Tasks

1. Open task details
2. Go to the Share tab
3. Enter collaborator's email
4. Set role: Viewer, Editor, or Admin

### Comments & Mentions

- Add comments below tasks
- Mention teammates with @name
- Receive notifications for mentions

## Settings

### Theme Customization

- Choose from preset themes
- Customize colors
- Save to your profile

### Keyboard Shortcuts

- View all shortcuts with ?
- Customize in Settings > Keyboard

## Mobile App

TaskPlanner is available as a Progressive Web App (PWA):

1. On mobile browser, tap "Install"
2. Add to home screen
3. Works offline with sync

## Troubleshooting

### Common Issues

**Tasks not syncing?**
- Check internet connection
- Refresh the page
- Check sync indicator in toolbar

**Can't create task?**
- Ensure you're logged in
- Check database permissions

**Missing features?**
- Enable in Settings
- Check your role/permissions
